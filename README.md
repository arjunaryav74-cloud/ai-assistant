# AI Assistant

## Goal
A chat-first personal AI assistant that feels like one brain.

## V1 does
- Chat with the user
- Save conversation history
- Store useful long-term memories
- Let the user ask things in natural language

## V1 does not do
- Full Mac control
- Autonomous file cleanup
- Advanced voice workflows
- Complex tool ecosystems

## Stack
- Next.js
- Anthropic / Claude
- Supabase
- Cursor

## First tools
- Memory save
- Memory retrieval
- Reminders / tasks

## Database plan
- users
- conversations
- messages
- memories

## Later
- Voice
- Model switching
- Native Mac helper
- More tools
